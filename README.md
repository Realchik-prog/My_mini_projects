# Мои мини-проекты на Python

Здесь собраны небольшие программы и скрипты, которые я написал в начале своего пути в программировании. 
Они помогли мне освоить базовый синтаксис, алгоритмы и логику. 
Также я буду выкладывать сюда и свои будущие маленькие проекты

## 📂 Структура

Проекты сгруппированы по темам:

- **`games/`** — игры: угадай число, камень-ножницы-бумага, бросание кости, угадай еду...
- **`generators/`** - генераторы: цвета, пароля, числа...
- **`math/`** — математические алгоритмы: НОД/НОК, числа Фибоначчи, простые числа, счёт времени...
- **`utils/`** — полезные утилиты: URL-загрузчик, шифр Цезаря, логин-пароль, количество слов...

## 🚀 Как запустить

### Для запуска файлов-скриптов 
Достаточно выполнить:
```bash
python путь/к/файлу/от/корня/репозитория.py
```

### Для проектов с зависимостями (На данный только `url_loader`)

1. Создайте виртуальное окружение в корне репозитория:
```bash
python -m venv venv
```

2. Активируйте его:
```bash
source venv/Scripts/activate    # Windows
source venv/bin/activate        # Linux/macOS
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Запустите нужный проект:
```bash
python путь/к/файлу/от/корня/репозитория/main.py
```
